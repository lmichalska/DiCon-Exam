### 🎵 Image to Sound Converter (React App)

A React-based web application that converts images into playable sound by analyzing pixel data and mapping it to audio frequencies. The project is deployed using GitHub Pages.


### 🚀 Live Demo
👉 https://lmichalska.github.io/DiCon-Exam/

### 🧠 How It Works
The image is loaded into an HTML canvas
Pixel data (RGB values) is extracted
Pixel values are mapped to sound frequencies
The result is playable synthesized sound

### 📦 Installation
Install dependencies:
npm install

Run locally:
npm start

App runs at:
http://localhost:3000

### 🌐 Deployment (GitHub Pages)
npm run build
npm run deploy
